Detailed Description

template<typename T = int64_t, bool HeapData = (sizeof(T) > BLOCK_SIZE-sizeof(int64_t)*2-sizeof(int64_t*))>
struct Grappa::Vertex< T, HeapData >

Vertex with customizable inline 'data' field.

Will attempt to pack the provided type into the block-aligned Vertex class, but if it is too large, will heap-allocate (from locale-shared heap). Defines a '->' operator to access data fields.

Example subclasses:

// Vertex with parent
// Preferred:
struct Parent { int64_t parent; };
Vertex<Parent> p;
p->parent = -1;
struct VertexP : public Vertex<int64_t> {
VertexP(): Vertex() { parent(-1); }
int64_t parent() { return data; }
void parent(int64_t parent) { data = parent; }
};
