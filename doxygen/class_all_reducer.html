Detailed Description

template<typename T, T(*)(const T &, const T &) ReduceOp>
class AllReducer< T, ReduceOp >

A Reducer object encapsulates a reduction during the computation to a single location.

Warning: may only use one object of type AllReducer<T, ReduceOp> concurrently because relies on Grappa::allreduce, which has no way of tagging messages
