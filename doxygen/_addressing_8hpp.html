typedef int Pool

Global Addresses for Grappa.

We support two types of global addresses:

  1. 2D addresses
  2. Linear addresses

2D addresses are a PGAS-style tuple of ( core, address on core )

Linear addresses are block cyclic across all the cores in the system.
